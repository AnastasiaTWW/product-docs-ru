# Общие настройки сканера

Для перехода к настройкам сканера зайдите на вкладку *Сканер* в веб‑интерфейсе
и нажмите на кнопку *Настройки* слева.

![!Настройки сканера][img-configure-scanner]


## Сканер

Данная настройка позволяет включить или выключить сбор периметра и поиск типовых уявзимостей. 

Чтобы перезапустить сканирование периметра вручную, выключите настройку, если она включена, и снова включите ее.

!!! info "Просмотр найденных уязвимостей"
    Найденные сканером уязвимости можно просмотреть на вкладке [*Уязвимости*][link-vulnerabilities] веб‑интерфейса.

В настройках сканера вы можете изменять его конфигурацию:
* Настраивать список обнаруживаемых уязвимостей;
* Включать или отключать повторную проверку уязвимостей.

!!! info "Настройка сканера"
    Подробная информация о настройке сканера доступна по этой [ссылке][link-scanner-modules].

## Активная проверка атак

Эта настройка позволяет полностью отключить автоматическое воспроизведение
атак сканером. Вы сможете в дальнейшем вручную запустить воспроизведение атаки,
нажав на кнопку *Проверить*, или на вкладке *Правила*.

## Ограничение на RPS сканера

Эта настройка позволяет ограничить максимальную нагрузку на веб‑приложение,
которую будут создавать запросы от сканера.

Можно настроить максимальное количество запросов в секунду (RPS):
* на домен;
* на IP‑адрес.

Если с одним IP‑адресом связано несколько доменов, то суммарная по всем доменам скорость запросов к этому
IP‑адресу не будет превышать ограничения для IP‑адреса. Если с одним доменом
связано несколько IP‑адресов, то суммарная скорость запросов к этим IP‑адресам
в рамках этого домена не будет превышать ограничения для домена.

Установленные настройки сканера применяются глобально для всех доменов и IP‑адресов в периметре. Переопределить эти ограничения для отдельных IP‑адресов или доменов можно в [сетевом периметре][link-scope-rps-limit].

Чтобы вернуть ограничения на RPS сканера по умолчанию, нажмите на кнопку *По умолчанию*.

----------

Чтобы принять внесенные в настройки сканера изменения, нажмите на кнопку *Сохранить.*.
