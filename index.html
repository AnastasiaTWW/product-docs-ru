    Главная

    Валарм WAF обладает неограниченным горизонтальным масштабированием и защищает веб-приложения и API от угроз OWASP Top 10, ботов, атак на логику приложения и других угроз.

    Быстрый старт

    Установите первую WAF‑ноду как модуль NGINX за 20 минут

    Обзор Валарм WAF

    Изучите возможности Валарм WAF, архитектуру продукта и набор поддерживаемых платформ

    Установка

    Выберите платформу и разверните первую WAF‑ноду в ифраструктуре вашего приложения

    Настройка

    Расширьте возможности Валарм WAF с кастомными настройками развернутой WAF‑ноды

    Валарм API

    Узнайте об использовании Валарм API для расширения функциональности Валарм WAF

    Управление WAF через интерфейс

    Отслеживайте состояние безопасности вашей системы и управляйте WAF‑нодой вместе с командой через Консоль управления Валарм
